# MetricFlow PRONABEC — Dashboard de Beneficiarios

Dashboard interactivo de beneficiarios PRONABEC. PWA instalable en Android e iOS.

## 🚀 Ver en vivo
👉 **[Abrir Dashboard](https://TU-USUARIO.github.io/pronabec-dashboard/)**

## 📱 Instalar como App
- **Android**: Abre el enlace en Chrome → menú ⋮ → "Agregar a pantalla de inicio"
- **iPhone/iPad**: Abre en Safari → □↑ Compartir → "Agregar a pantalla de inicio"

## 📊 Uso
1. Abre el dashboard
2. Carga tu archivo CSV con el botón **Cargar CSV**
3. Explora los gráficos y filtros

## 🛠 Tecnologías
- HTML5 + CSS3 + JavaScript puro
- Chart.js para gráficos
- PWA (Progressive Web App) instalable
